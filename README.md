# ArcaneTools
Arcane Odyssey Tools by jhxu/amechanos. Use to your own discretion and read the official in-game [rules](https://docs.google.com/document/d/1oh7TLdW6aN6mAyk55BDF_ga-eMl_nHNECo2Bmqa71XA/edit?usp=sharing) to minimise account bans or deletion. 

## To launch and run autofish:

- Make sure you have the latest version of Python installed.
- Install all dependencies by running ```pip install pyautogui keyboard pillow```
- Run the file in either command prompt via ```python path/to/the/file/autofish.py``` or run it directly in a code editor like Visual Studio Code.
- In line 39 is the hotkey to stop the automation process.
- In line 40 is your fishing rod hotkey (your rod should be in your hotbar). Change this accordingly.
- Default rod key is 6 and the slot switch key is 1.
- Automated eating isn't included.
